# 龙之旅行

|         |                        |
| :-----: | :--------------------: |
| 适用服务端核心 |        1.12.2全核心       |
|   前置插件  | DragonCore，ProtocolLib |
|   可选前置  |     PlaceholderAPI     |

## 1.插件简介

1.支持在线设置路径多种路径配置。

2.支持海陆空三种旅行方式，增加你的乐趣同时，减少玩家跑路时间。

3.支持自定义图片显示目的地，可以配置动画和显示的位置，全部自定义。

4.自定义旅行器模型，支持多种模型自由搭配，搭配多种速度与权限。

5.支持起飞事件触发与落地事件触发，完全自定义你的命令。
